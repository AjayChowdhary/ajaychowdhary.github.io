PROJECTS


IEEE-NSIT App

IEEE-NSIT Official App :Created an Android platform for I.E.E.E. (NSIT) to connect 150+ members each year. The Application connects the user with the IEEE society for getting daily feeds, event updates, project description and Special Interest Groups update. The app registers itself in our online database using app token.

Health Kit

A project based on IOT to check heart rate, body temperature etc in real time and send data to cloud machine.We used AWS EC2 machine,S3 container service. The data from circuit is uploaded using Arduino uno. The uploaded content is converted into JSON and is provided to the user.A webapp and an android app are made to keep track of health parameters and notify and online consult doctor through chat forum if found unfit.

Attendance tracker

This App is made to keep a track of attendance/ event. The produces a notification in the main activity of the user so that user can mark their review on that event. A proper database is maintained for the user to check out the past attendance. This app targets every college individual who wants to keep track of their attendance in a proper manner.

Music player

An Android Application that intelligently filters the mp3 files in your device memory and plays it for the user. The App provides a great UI with the use of Viewpager and a sleep timer feature that enables the user to decide the pausing or stopping/pausing time for the app.

Code Practice Oscillator

NOV 2015

A device to send and receive morse code. ​ The project uses 555 timer IC configured as an astable mode which is a simple way to practice the sending and receiving of Morse Code.

Power Supply

Dec 2015

Made a Variable Power Supply with short circuit protection which generated both positive and negative volatages ranging from 1.5V to 37V and a maximum current of 3A.Made for use in other projects. Board is fabricated using toner transferred method.
